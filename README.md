# Dayne Bonuses Viewer

Casino bonus data dashboard. Freshly scraped bonus offers displayed with sortable tables.

**Live site:** https://gemquota.github.io/dayne-bonuses/

## Data

- `public/dayne-bonuses.csv` — Raw scrape output
- `public/dayne-bonuses-cleaned.csv` — Filtered/cleaned version

## Build

```bash
npm install
npm run build
```

Deploys automatically via GitHub Actions on push to `main`.
